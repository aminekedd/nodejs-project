To get started with the project, follow these steps:

Clone the repository:
git clone https://github.com/aminekedd/nodejs-project
Install the dependencies for the backend API:
cd booking-project/api
npm install
Install the dependencies for the frontend:
cd booking/client

npm install

cd booking/admin

npm install

Start the backend API:
cd booking/api
npm start
Start the frontend:
cd booking/client
npm start
The frontend should now be running at http://localhost:3000 and the backend API should be running at http://localhost:8000.
